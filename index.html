  • Technical Projects

    Mind the Gap

    An Android Application…

    Mind the Gap is an Android application that displays real time information of the London Underground including GPS location detection, train arrival times, and nearest stations.

    Implementation

    Using JSON parsing, the application extracts live updates on the train arrivals, station locations, and underground lines. Using the OSMDroid and OSMBonusPack library from Java in conjunction with the parsed data, a map with tube lines and tube stations was created to ensure a user-friendly interface.

    Relevant Programs

    The backend portion of the application was developed using Java inside the IntelliJ IDEA development environment. Testing of the application was performed with both the Genymotion Android Emulator and JUnit tests in Java.

    MDHelper

    A Web Application...

    MDHelper enhaces the productivity of physicians by allowing patients to send medical information before appointments so that physicians can focus on diagnosis and treatment during the consultation

    Implementation

    Utilized HTML5, CSS and JavaScript to implement the multi-page application and used IronRouter to navigate through pages. The Reactive FullCalendar API was used to enable online appointment scheduling. Also integrated MongoDB to store patient and appointment data.

    The front end was developed using HTML, CSS and JavaScript while the back end was implemented using the Meteor.js framework which connects seamlessly with MongoDB. Iron Router was used to navigate through the pages and FullCalendar API was integrated allow appointment scheduling.

  • Technical Projects

    StudyBuddy

    A Chrome Extension…

    Studybuddy is a chrome extension that blocks distracting websites so users can focus on important tasks at hand.

    Implementation

    The extension is formed by a collection of Javascript, JSON, and HTML files that works asynchronously to accept URLs and inputs and then blocks the specified URLs in the background. Other simple user interactions include turning the extension on and off through a popup in the toolbar.

    Relevant Programs

    Using JavaScript Object Notation (JSON) parsing and chrome.local storage in the Chrome API, information was successfully transferred between files to allow for user interaction. Other Chrome APIs such as chome.webRequest and chrome.events were used to implement webpage blocking.

    A Game Application...

    Dino is a Java game application in which players try to explore the game world and collect all the items in as few steps as possible.

    Accomplish and Reward

    An Android Application…

    Reward and Accomplish is an Android Application that connects performance with reward by allowing users to assign a reward to be achieved after items on the to-do list have been completed.

    Implementation

    By building on the layout files through design and text editors, a collection buttons work together to smoothly transition from one page to the next.   • Technical Projects

    Missile Madness

    A Python Game...

    Missile Madness is a Pygame application in which enemy UFOs descend from the top of the screen and must be destroyed with a missile before it reaches the bottom of the screen.

    Implementation

    This Pygame application is dependent on a single for-loop that ticks and renders the game display on each clock cycle. The implementation of sprite classes within the program enabled collision detection and sprite spawning while increasing the robustness of the code.

    Relevant Programs

    This game was coded inside the PyCharm EDU coding environment using Pygame, an extension of Python especially designed for coding games. Throughout the program, the Mixer, Image, and Sprite libraries of Pygame were used to manage and render sprites, detect sprite location, and add sound effects.
