    StudyBuddy


    View More     x

    A Chrome Extension…

    Studybuddy is a chrome extension that forces users to focus and tackle their tasks without the hindrance of distracting websites.


    View More     x

    Implementation

    The extension is formed by a collection of Javascript, JSON, and HTML that works asynchronously to accept URLs and inputs and then blocks the specified URLs in the background. Other simple user interactions include turning the extension on and off through a popup in the toolbar.


    View More     x

    Relevant Programs

    Using JavaScript Object Notation (JSON) parsing and chrome.local storage in the Chrome API, information was successfully transferred between files to allow for user interaction. Other Chrome APIs such as chome.webRequest and chrome.events were used to implement webpage blocking.
